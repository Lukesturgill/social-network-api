# Social Network API

## Purpose
Social Network API is a backend API that uses mongoDB to allow the server to add/remove/update users, thoughts, reactions to thoughts, and friend connections.

## Built With
* Mongoose
* Express.js
* JavaScript

## Installation 
* `npm start`
* `open http://localhost:3001/api/ in Insomnia app.
